\subsection{Detailed Description}
Character string manipulation methods. 

These methods work generally on character strings, but also, more particularly on lists. A list is a character string that contains substrings separated by a delimeter. That delimeter can be set by the user but the default is \char`\"{}\+:\char`\"{}. Colon delimeted lists are used in O\+A\+S\+I\+S and M\+C\+T mainly to instantiate a list of fields, such as \char`\"{}temperature\+:humidity\+:zonal\+\_\+velocity\+:meridiona\+\_\+velocity\char`\"{}.
